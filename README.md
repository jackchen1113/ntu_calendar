# ntu_calendar
台大行事曆試作_eva01

## 使用技術

- **HTML**: 基礎結構
- **Alpine.js**: 輕量級的JavaScript框架，用於響應式互動
- **Day.js**: 處理日期和時間的輕量庫
- **Tailwind CSS**: 美化外觀的CSS框架

## 組件架構

```
<Calendar>
 ├── <CalendarHeader>（月份切換、今天按鈕）
 ├── <CalendarGrid>（7x6日期格）
 │     ├── <CalendarCell>（單一天格，顯示日期，顯示圓點為當日有事件）
 └── <EventModal>（查看事件）
```

## 功能說明

### CalendarHeader
- 顯示當前年月
- 提供前一個月/下一個月切換按鈕
- 「今天」按鈕快速返回當前日期

### CalendarGrid
- 7列（週日到週六）x 6行的日期格
- 自動計算每月的日期顯示
- 包含上個月和下個月的日期（灰色顯示）

### CalendarCell
- 顯示日期數字
- 當天日期以藍色背景高亮顯示
- 有事件的日期顯示藍色圓點（最多顯示3個）
- 點擊可開啟事件詳情

### EventModal
- 點擊日期後彈出的事件詳情視窗
- 顯示該日期的所有事件
- 包含事件標題和描述
- 點擊外部或關閉按鈕可關閉視窗

## 資料來源

行事曆資料載入自固定的 `events.json` 檔案，包含台大相關的重要日期和活動。

## 如何使用

1. 開啟 `index.html` 檔案即可在瀏覽器中查看行事曆
2. 無需安裝任何依賴，所有資源都通過CDN載入
3. 可以直接部署到任何靜態網站託管服務

## 檔案說明

- `index.html`: 主要的HTML檔案，包含所有組件和邏輯
- `events.json`: 事件資料檔案，包含所有行事曆事件
